# Agentic workflow basics

## summary
agentic workflows split a problem into clear stages so each stage can be optimized and validated independently.

## answer
the master agent owns control flow and delegates focused tasks to specialized agents.
each agent returns structured output so downstream steps remain predictable.
practical systems should preserve trace logs and compact artifacts after each run.
